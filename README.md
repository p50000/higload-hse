# Проектирование высоконагруженного сервиса: Агрегатор авиабилетов
## 1. Тема и целевая аудитория
### 1.1 Тема проекта 
Метапоисковик (агрегатор) авиабилетов и туристических услуг.
Проектируемая система является платформой, которая позволяет пользователям искать авиабилеты по различным направлениям и датам. Сервис агрегирует предложения от сотен авиакомпаний и онлайн-трэвел агентств (OTA), предоставляя пользователю сводную информацию и перенаправляя его на сайт партнера для совершения покупки.


### 1.2 Реальные аналоги 
- Aviasales
- Яндекс Путешествия
- Купибилет

## 2. Целевая аудитория

### 2.1 Характеристика аудитории
Российские пользователи в возрасте от 25 до 45 лет, 50% женщин, 50% мужчин. [[2]](https://mediakit.aviasales.ru/)

### 2.2 Размер аудитории и нагрузки

- **MAU:** 20 миллионов человек ежемесячно [[1]](https://www.aviasales.ru/psgr/article/aviaseils-eto-bezopasno)
- **150 000 000** поисков билетов. в день [[2]](https://mediakit.aviasales.ru/)

### 2.3 Местоположение
Россия (все регионы).

## 3. Функционал MVP
Для проектирования выделен следующий ключевой функционал, составляющий ядро сервиса:

- **Поиск авиабилетов по заданным параметрам**: Пользователь вводит города вылета и прилета, даты, количество пассажиров. Система инициирует поиск по базам партнеров.
- **Отображение результатов поиска**: Вывод списка найденных билетов в стандартизированном виде. Реализация базовой сортировки (по цене, времени в пути) и фильтрации (по пересадкам, авиакомпаниям).
- **Переход на сайт партнера для бронирования**: Ключевая функция агрегатора. После выбора предложения система перенаправляет пользователя на сайт авиакомпании или агентства для оформления и оплаты билета.
- **Кэширование поисковых запросов**: Для снижения нагрузки на API партнеров и уменьшения времени ответа для пользователя, результаты популярных запросов кэшируются на короткое время.

**Ключевые продуктовые решения:**
- **Модель метапоисковика**: Сервис не является продавцом. Ключевое архитектурное решение — система не хранит информацию о бронированиях и не обрабатывает платежи.
- **Приоритет скорости над актуальностью цен**: Ввиду высокой волатильности цен на авиабилеты, допускается использование краткосрочного кэша (с временем жизни в несколько минут). Цена может незначительно отличаться от финальной на сайте партнера.
